文章插入图片

图床：完成图片的插入，这种网站就被成为图床。当博文中有图片时，若是少量图片，可以直接把图片存放在source文件夹。或者把主页配置文件_config.yml 里的post_asset_folder:这个选项设置为true，在你的hexo目录下执行这样一句话npm install hexo-asset-image –save，这是下载安装一个可以上传本地图片的插件，等待一小段时间后，再运行hexo
